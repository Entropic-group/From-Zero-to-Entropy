# The Programming GYM

As you probably already heard many (many!) times before, practicing is basically a prerequisite to being successful in programming. With that in mind, you're welcome to our Programming Gym. The list below contains many resources that allow you to practice programming, from LeetCode-style problems to full project ideas.

- [Exercism](https://exercism.io/)
- [LeetCode](https://leetcode.com/)
- [HackerRank](https://www.hackerrank.com/)
- [CodeWars](https://www.codewars.com/)
- [100+ Python Coding Problems with Solutions](https://github.com/ProgrammingHero1/100-plus-python-coding-problems-with-solutions)
- [Rosalind](http://rosalind.info/problems/locations/)
- [Advent of Code](https://adventofcode.com/)
- [GitHub projects for beginners](https://github.com/MunGell/awesome-for-beginners)
